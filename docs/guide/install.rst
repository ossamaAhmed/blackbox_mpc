.. _install:

===============
Installation
===============

---------------------------------------
Install the latest release through  pip
---------------------------------------

.. code-block:: shell

   pip install blackbox_mpc

-------------------
Install from source
-------------------

Clone this repo and then install the package

.. code-block:: shell

   git clone https://github.com/ossamaAhmed/blackbox_mpc
   cd blackbox_mpc

Install the blackbox_mpc package.

.. code-block:: shell

   pip install -e .
